# PuzzGame
O PuzzGame é uma versão moderna do clássico Tetris, desenvolvida para testar lógica, reflexos e estratégia. O jogador deve encaixar blocos que caem, formando linhas completas para pontuar e evitar que a tela se encha. Criado em HTML, CSS e JavaScript, o jogo combina nostalgia e diversão em um desafio envolvente.
